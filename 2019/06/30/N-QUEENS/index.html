题目

n 皇后问题研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。

上图为 8 皇后问题的一种解法。

给定一个整数 n，返回所有不同的 n 皇后问题的解决方案。

每一种解法包含一个明确的 n 皇后问题的棋子放置方案，该方案中 ‘Q’ 和 ‘.’ 分别代表了皇后和空位。

示例:

输入: 4
输出: [
 [".Q..",  // 解法 1
  "...Q",
  "Q...",
  "..Q."],

 ["..Q.",  // 解法 2
  "Q...",
  "...Q",
  ".Q.."]
]

总结

  1. 深度优先搜索 DFS + 剪枝 （遍历+减少无效的遍历）
  2. 列、撇、捺 分为三个数组进行缓存，减少无效的遍历
  3. 撇和捺连个数组的大小为什么是4*n-1？？？
  4. 每一行的输出可以通过以下方式输出
    • char[] chars = new char[n];Arrays.fill(chars,’.’);chars[i]=’Q’;String s = new String(chars);
  5. 递归时候首先放置了queen的位置，在回退的时候也需要将位置还原，才能继续之后的递归。
  6. 不用每次都开辟新的空间存贮，可以通过clone（）方法实现
