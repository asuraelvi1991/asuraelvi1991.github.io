REDIS

https://hub.docker.com/_/redis

  1. 下载redis docker pull redis

  2. 运行redis docker run –name redis -d -p 6379:6379 redis

  3. 查询所有docker docker ps -a

  4. 如果已经运行了redis，可以直接docker start redis

  5. 包 jedis

  6. @Bean jedisPool(host), jedisPool.getResource()–>jedis,操作：jedis.hset\jedis.hget

  7. redis哨兵模式 jedisSentinelsPool

  8. redis集群模式 jediscluster数据分片

  9. spring 缓存抽象

    • 为java方法增加缓存缓存执行结果
    • 支持多种类型
    • 接口 Cache CacheManager

  10. 使用缓存的不同场景

    • 长久时间不会变，可以有时延，jvm内部缓存
    • 具备一定一致性 ，分布式缓存 redis
    • 不使用缓存，写一次读一次，不需要使用缓存

  11. 引入redis的包 starter-data-redis\starter-cache\apach.commons-pool2

  12. redis template
